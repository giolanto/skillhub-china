# SkillHub China

中国首个针对OpenClaw和渠道机器人的开源技能市场

## 功能特性

- 🎯 技能展示与搜索
- 📦 支持多渠道（飞书、微信、钉钉、Telegram）
- 🚀 一键安装到OpenClaw
- ⭐ 评分与评论系统
- 🔄 GitHub自动同步

## 技术栈

- Next.js 14
- Tailwind CSS
- Supabase (PostgreSQL)
- Vercel

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 部署

1. 推送到GitHub
2. 导入Vercel
3. 配置环境变量
4. 完成！
